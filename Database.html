SQL - Employee Database

Design a database system to store and manage employee records efficiently.

Project Overview

In this project, you will design a simple employee database using SQL. Learn to create tables, insert records, and run basic queries.

Tools & Skills Needed

  • SQL knowledge (CREATE, INSERT, SELECT, UPDATE, DELETE)
  • Database management system (e.g., MySQL, SQLite, PostgreSQL)

Step-by-Step Instructions

  1. Setup the Database: Create a database called company.
  2. Create the Employee Table: Use the following SQL commands:
    CREATE TABLE employees (
      id INT PRIMARY KEY AUTO_INCREMENT,
      name VARCHAR(100),
      department VARCHAR(50),
      salary DECIMAL(10, 2)
    );
  3. Insert Records: Insert a few sample employees:
    INSERT INTO employees (name, department, salary) VALUES
    ('Alice Johnson', 'HR', 55000),
    ('Bob Smith', 'Engineering', 75000),
    ('Carol Lee', 'Sales', 60000);
  4. Query the Table: Retrieve employee records:
    SELECT * FROM employees;
  5. Update or Delete Records: Try updating a salary:
    UPDATE employees SET salary = 58000 WHERE name = 'Alice Johnson';
    Or delete a record:
    DELETE FROM employees WHERE name = 'Bob Smith';

Expected Outcome

A functional employee database where you can store and manage employee data using SQL queries.
