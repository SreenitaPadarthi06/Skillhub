Python - Weather App

A step-by-step guide to build a simple weather application using Python and API integration.

Project Overview

This project creates a Python program that fetches live weather data from an online API and displays it. You will learn how to work with APIs, handle JSON data, and basic Python programming.

Tools & Skills Needed

  • Python 3 installed on your computer
  • Basic Python programming (variables, functions, libraries)
  • Internet access to use a weather API (e.g., OpenWeatherMap)

Step-by-Step Instructions

  1. Install Requests library: Open your terminal and run: pip install requests
  2. Get an API Key: Register at OpenWeatherMap and get your free API key.
  3. Write the Python script: Use the code below to fetch and display weather data:
    import requests

    api_key = "YOUR_API_KEY"

    url = f"http://api.openweathermap.org/data/2.5/weather?q={city}&appid={api_key}&units=metric"

    response = requests.get(url)

  5. Optional Enhancements: Add error handling, display more details, or build a GUI using Tkinter.

Expected Outcome

Your program will display the current temperature and weather description for any city entered.
