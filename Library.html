Java - Library Management System

A step-by-step guide to building a basic Library Management System in Java.

Project Overview

This project will teach you how to create a simple system to manage a collection of books in a library. You'll learn how to use Java's core concepts like classes, objects, and collections to build a functional program.

Tools & Skills Needed

  • Java JDK installed on your computer
  • Basic understanding of Java (variables, classes, objects, lists)
  • A text editor or IDE (like Eclipse, IntelliJ IDEA, or VS Code)

Step-by-Step Instructions

  1. Set up your project: Create a new Java file named LibraryManagement.java.
  2. Define the Book class: This class will store book details (ID, title, author, etc.).

    class Book {
      int id;
      String title;
      String author;

      public Book(int id, String title, String author) {
        this.id = id;
        this.title = title;
        this.author = author;
      }

      public String toString() {
        return id + " - " + title + " by " + author;
      }
    }
  3. Create the main program: In your main method, create a list to store books and provide options to: - Add a book - View all books - Update book details - Delete a book

    import java.util.*;

    public class LibraryManagement {
      public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        ArrayList<Book> books = new ArrayList<>();

        while (true) {
          System.out.println("
    1. Add Book
    2. View Books
    3. Update Book
    4. Delete Book
    5. Exit");
          System.out.print("Enter choice: ");
          int choice = sc.nextInt();

          switch (choice) {
            case 1:
              System.out.print("Enter ID: ");
              int id = sc.nextInt();
              sc.nextLine();
              System.out.print("Enter Title: ");
              String title = sc.nextLine();
              System.out.print("Enter Author: ");
              String author = sc.nextLine();
              books.add(new Book(id, title, author));
              System.out.println("Book added!");
              break;

            case 2:
              System.out.println("Books in Library:");
              for (Book b : books)
                System.out.println(b);
              break;

            case 3:
              System.out.print("Enter ID to update: ");
              int updateId = sc.nextInt();
              sc.nextLine();
              for (Book b : books) {
                if (b.id == updateId) {
                  System.out.print("Enter new title: ");
                  b.title = sc.nextLine();
                  System.out.print("Enter new author: ");
                  b.author = sc.nextLine();
                  System.out.println("Book updated!");
                  break;
                }
              }
              break;

            case 4:
              System.out.print("Enter ID to delete: ");
              int deleteId = sc.nextInt();
              books.removeIf(b -> b.id == deleteId);
              System.out.println("Book deleted!");
              break;

            case 5:
              System.out.println("Goodbye!");
              return;

            default:
              System.out.println("Invalid choice.");
            }
        }
      }
    }
  4. Run your program: Compile and run the program in the terminal or your IDE to test adding, updating, and deleting books.
  5. Optional Enhancements: Add search functionality, save data to a file, or create a GUI with Java Swing.

Expected Outcome

Your program will allow you to manage books in the library through a console-based interface.
